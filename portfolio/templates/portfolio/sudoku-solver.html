Sudoku Solver

This is a simple sudoku solver that can solve most of sudoku problem (except "Expert/Very Hard" level).

Insert number in the corresponding square block, and click "SOLVE" button.

The process is to send the unsolved sudoku to backend script, and reply the solved sudoku to frontend through Django.

Completed code can be found in Github through here for Python as backend and here for javascript as frontend.
